Browser port of WernerAGA 1.0 (Patrick Klie, 21.5.1995) — a 68k assembler game for Amiga OS3 / AGA. Rules, levels, palette, sprites and title screen are taken straight out of src/Werner.asm.

Unofficial non-commercial fan preservation project. This game was originally created by Patrick Klie in 1995 as a private teenage programming project and was ported to the browser in 2026.
